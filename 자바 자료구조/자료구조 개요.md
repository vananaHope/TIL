# 자료구조 개요
* 자료구조는 다양한 자료를 효율적으로 표현하고 저장하고 처리하여 사용할 수 있도록 하는 것

## 자료구조의 분류
* 단순구조
  * 정수
  * 실수
  * 문자
  * 문자열
* 선형구조
  * 리스트
  * 연결 리스트
  * 스택
  * 큐
  * 덱
* 비선형구조
  * 트리
  * 그래프
* 파일구조
  * 순차파일
  * 색인파일
  * 직접파일

## 자료의 표현
* 컴퓨터는 이진수 코드를 기본적으로 사용
* 한 자리의 1 또는 0을 표현하는 단위를 **비트(bit)** 라고 한다.
* 비트는 자료를 표현하는 최소 단위
* 4개의 비트 그룹은 **니블(nibble)** 8개의 비트 그룹은 **바이트(byte)**
